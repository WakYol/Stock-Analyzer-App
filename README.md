## 🔍 Overview

A real-time stock analysis platform built with **ViteJS** (frontend) and **Node.js** (backend).  
It fetches live market data via the **Yahoo Finance API**, runs deep algorithmic calculations in the background, and provides users with technical indicators + an **experimental final trading signal**.

---

## ✨ Key Features

- 📡 **Real-time stock data** from Yahoo Finance  
- 📊 **Technical indicators** (RSI, MACD, Moving Averages, Bollinger Bands)  
- 🧠 **Deep background computation** – heavy math processed without blocking UI  
- 🔮 **Experimental final signal** – aggregated buy/sell/hold suggestion  
- ⚡ **Fast UI** powered by ViteJS
